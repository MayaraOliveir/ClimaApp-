<h1 align="center">Clima Agora</h1>

Um aplicativo simples e moderno para consultar a temperatura atual de uma cidade, construído com tecnologias web modernas e sem dependências externas. Este projeto é destinado a fins educacionais e foi desenvolvido com a ajuda de Inteligência Artificial.

## Visão Geral

Este projeto é um aplicativo web que permite aos usuários pesquisar a temperatura atual de uma cidade usando a API [Open-Meteo](https://open-meteo.com/). Ele inclui funcionalidades como validação de entrada, geocoding, previsão do tempo e estados de carregamento/erro. O projeto foi desenvolvido para fins educacionais e demonstra o uso de JavaScript moderno, HTML5, CSS3 e APIs públicas.

## Funcionalidades

- **Consulta de Temperatura Atual**: Digite o nome de uma cidade e obtenha a temperatura atual.
- **Validação de Entrada**: Garante que a entrada do usuário seja válida e normalizada.
- **Estados de Carregamento/Erro**: Exibe mensagens de carregamento e erro para melhor experiência do usuário.
- **Geocoding**: Converte o nome da cidade em coordenadas geográficas usando a API de geocoding do Open-Meteo.
- **Tema Visual**: Altera o tema visual do aplicativo com base nas condições climáticas atuais.
- **Acessibilidade**: Inclui recursos de acessibilidade, como rótulos ocultos e mensagens ao vivo.

## Pré-requisitos

- Navegador moderno com suporte a JavaScript e CSS3.
- Conexão à internet para acessar a API do Open-Meteo.

## Como Usar

1. **Clone o Repositório**:
   ```bash
   git clone https://github.com/MayaraOliveir/ClimaApp-
   ```

2. **Abra o Projeto**:
   - Navegue até a pasta do projeto:
     ```bash
     cd App-Clima
     ```
   - Abra o arquivo `index.html` no seu navegador.

3. **Digite o Nome de uma Cidade**:
   - No campo de entrada, digite o nome de uma cidade e clique no botão "Buscar" ou pressione Enter.

4. **Veja os Resultados**:
   - A temperatura atual da cidade será exibida, junto com uma descrição do clima e um tema visual correspondente.

## Estrutura do Projeto

- **`index.html`**: A página principal do aplicativo.
- **`app.js`**: O script principal que controla a lógica do aplicativo.
- **`styles.css`**: O arquivo de estilo do aplicativo.
- **`weatherApi.js`**: Um utilitário para interagir com a API do Open-Meteo e processar dados climáticos.
